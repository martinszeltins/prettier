#### Recognize `<search>` element (#14615 by @fisker)

HTML spec added [`<search>` element](https://html.spec.whatwg.org/multipage/grouping-content.html#the-search-element).

<!-- prettier-ignore -->
```html
<!-- Input -->
<SEARCH title="Website">
  ...
</SEARCH>

<!-- Prettier stable -->
<SEARCH title="Website"> ... </SEARCH>

<!-- Prettier main -->
<search title="Website">...</search>
```
